# Semillero de Investigación SIIC
 
A static website created for the **PPI (Proyecto de Práctica/Proyectos de Investigación)** project at **Politécnico Colombiano Jaime Isaza Cadavid**.
 
## About the Project
 
This site was built for the *Semillero de Investigación en Inteligencia Computacional* (Computational Intelligence Research Group), providing a web page to host and organize research-group materials such as presentations, tutorials, and supporting documents.
 
This repository contains the front-end (HTML/CSS) side of the SIIC project.
 
## Tech Stack
 
- **HTML** — page structure and content
- **CSS** — styling
## Project Structure
 
```
SIIC-Repositorio/
├── Html/    # HTML pages
├── Css/     # Stylesheets
├── img/     # Images used across the site
├── pdf/     # PDF documents (presentations, materials, etc.)
└── README.md
```
 
## Getting Started
 
Since this is a static site with no build step or backend dependencies, you can run it directly in a browser:
 
1. Clone the repository:
```bash
   git clone https://github.com/SamuelPerezCO/SIIC-Repositorio.git
   cd SIIC-Repositorio
```
 
2. Open any of the HTML files inside the `Html/` folder directly in your browser, or serve the folder locally, e.g.:
```bash
   python -m http.server 8000
```
   Then visit `http://localhost:8000` in your browser.
